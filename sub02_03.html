CAD Viewer

HTML5 웹 기반, AutoCAD 도면파일 (*.dwg, *.dxf)을 PDF파일로의 포맷 변환시스템

CAD Viewer(searchin view V2.0)은 AutoCAD 형식의 도면 파일(*.dwg, *.dxf)을 HTML5를 지원하는 웹브라우저에서 변환/열람할 수 있도록 지원하는 프로그램으로 도면, PDF 파일 등을 별도 뷰어나 해당 프로그램 없이 원클릭으로 열람할 수 있다.

주요 기능 목록
분류 기능설명
도면 업로드 도면 파일 업로드 기능

변환 AutoCAD 파일(*.dwg,*.dxf)의 pdf 변환

제품 구성

  • 01사용자가 파일전송도구를 이용해서 서버에 업로드 한 파일들 중 도면파일은 인크루넷 도면변환 서버로 변환요청한다.
  • 021번 과정에서 요청한 도면파일의 변환이 성공하면 변환완료된 PDF파일과 헤더정보를 변환 요청한 서버로 리턴해 준다.
  • 03변환 요청 작업을 관리하고 변환된 도면파일을 웹브라우저에서 확인이 가능하다.
