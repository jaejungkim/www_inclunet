Tera Transfer 주요기능

  • 대용량파일, 대량파일, 다중파일, 폴더단위 파일의 고속 전송
    • 100GB 단위의 대용량 파일, 100개 이상의 대량파일 고속 전송
    • 문서, 영상, 이미지 파일 등 다양한 형태의 다중파일 고속 전송
    • 압축이 필요 없는 폴더 단위의 전체 파일 고속 전송
    • 전송 중 파일손실이 발생할 경우 해당 부분을 찾아 무결성 파일 전송
  • 자동 이어 전송 / 파일전송 보안 / 파일전송 상황 모니터링 기능
    • 불안정한 네트워크 환경에서 회선 장애시 회선의 복구와 동시에 자동 복구/이어 전송
    • SSL/TLS 보안 프로토콜 지원
  • 다양한 서버 환경 및 개발 언어 지원
    • 크로스 브라우저 / 크로스 플랫폼 (리눅스, 윈도우, 맥)
    • HTML5가 지원되는 MS Edge, Chrom, Firefox, Safari 등 다양한 브라우저 지원
    • 위의 브라우저가 작동하는 모든 OS지원
  • 사용자 친화적 다양한 편의 기능
    • 웹표준, 웹접근성 준수 반응형 UI 구현 / 이미지파일의 썸네일 형식 미리보기 지원
    • 다양한 언어 지원 / 사용자 임의 파일전송 일시정지, 재개 지원
    • Drag&Drop 기능 지원 / 파일 송/수신 음성 및 메세지 알림 지원
  • 사용자 및 기관의 전송 정책에 따른 설정 지원
    • 전체 파일 최대용량, 파일별 최소/최대 크기 제한
    • MIME타입별 확장자 제한
    • 첨부파일 최소/최대 개수 제한 / 동시 다운로드 개수 제한 등
